紫微斗數排盤 API：輸入出生資料，取得本命盤與運限（大限／小限／流年／流月／流日／流時）之結構化 JSON 或純文字命盤。 所有端點同時支援 GET（query 參數）與 POST（JSON body，欄位同名）。回應皆為 UTF-8 繁體中文。 機器可讀版本（適合直接作為 LLM 工具描述）：GET /api/v1/doc。

共用參數（出生資料）

參數說明
calendarsolar 國曆（預設）或 lunar 農曆
year / month / day出生年月日（整數，依 calendar 解讀；支援 1901–2099）
isLeapMonth是否閏月（布林，僅農曆輸入時有意義，預設 false）
timeIndex出生時辰 0–12：0=早子(00–01)、1=丑(01–03)、2=寅(03–05)、3=卯(05–07)、4=辰(07–09)、5=巳(09–11)、6=午(11–13)、7=未(13–15)、8=申(15–17)、9=酉(17–19)、10=戌(19–21)、11=亥(21–23)、12=晚子(23–00)
hour亦可改傳出生鐘點 0–23（與 timeIndex 擇一）
gendermale/female 或 男/女

GET/POST/api/v1/natal — 本命盤

回傳完整本命盤 JSON：四柱、五行局、命主身主、生年四化、十二宮（主星／輔星／雜曜、亮度、四化、離心↓／向心↑自化、四組神煞、大限區間、小限歲數、身宮與來因宮標記）。

GET /api/v1/natal?year=2000&month=8&day=16&timeIndex=2&gender=female
宮位索引慣例：寅宮 = 0 順行至丑宮 = 11；亮度為廟／旺／得／利／平／不／陷；四化為祿／權／科／忌。

GET/POST/api/v1/horoscope — 本命盤＋運限

本命參數同上，另加目標日期，回傳 data.本命 與 data.運限（大限／小限／流年／流月／流日／流時，各層含宮位、干支、該層四化、流曜、宮名重排）。

參數說明
targetDate運限目標日 YYYY-MM-DD（國曆，預設今日）
targetTimeIndex目標時辰 0–12（流時用，預設 0）；或改用 targetHour（0–23）
GET /api/v1/horoscope?year=2000&month=8&day=16&timeIndex=2&gender=female
    &targetDate=2026-06-13&targetHour=10

GET/POST/api/v1/text — 純文字命盤（LLM 解盤用）

以樹狀純文字輸出完整命盤，可直接貼入任何 LLM 對話。內容包含：基本信息、命盤十二宮 （星曜亮度、生年四化、自化）、神煞、限流疊宮、十二大限共 120 流年列表， 以及依 layers 選擇的各運限維度盤。

參數說明
layers 輸出哪些運限維度盤（逗號清單）：decadal 大限、age 小限、yearly 流年、 monthly 流月、daily 流日、hourly 流時； 或 all（預設，全部）、none（僅本命盤）。中文名稱亦可。
每個維度各自輸出完整十二宮盤段：該層宮名重排、該層四化直接標記於星曜（如 [月忌]＝流月化忌）、該層流曜落宮。 建議把需要的維度一次列出，讓 LLM 直接閱讀疊盤，不要讓模型自行推算流月／流日／流時。
targetDate同上；傳 none 可完全省略運限段落
GET /api/v1/text?year=1991&month=8&day=6&hour=20&gender=male
    &targetDate=2026-06-13&layers=decadal,yearly,monthly

輸出片段示意

├流月盤［癸巳，流月命宮在寅宮］
│ ├流月四化 : 破軍化祿、巨門化權、太陰化科、貪狼化忌
│ ├十二宮（宮名以流月重排，括號內為本命宮名）
│ │ ├流月命宮[庚寅]（本命交友）｜星曜 : 廉貞[廟][↑忌],文曲[平][生年科]｜流月流曜 : 無
│ │ ├流月福德[壬辰]（本命疾厄）｜星曜 : 破軍[旺][月祿]｜流月流曜 : 月喜

排盤口徑

項目約定
年界以農曆正月初一換年（非立春）
晚子時timeIndex=12（23:00–00:00）採翌日子時法：日柱與日系星曜依翌日，月、年不變
閏月閏月十六（含）以後依下月安星（命宮與月系星曜），月柱與流月亦進位
大限起運歲數＝五行局數；陽男陰女順行、陰男陽女逆行
童限虛歲未達起運歲數時，依「一命二財三疾厄、四歲夫妻五福德」古訣取宮
流月斗君法：流年支宮起正月逆數至生月，再順數至生時
庚干四化太陽、武曲、太陰、天同
自化離心↓＝本宮宮干使本宮星曜化出；向心↑＝對宮宮干使本宮星曜化入
錯誤回應：參數錯誤回 400，JSON 端點為 {"success":false,"error":"…"}，文字端點為純文字錯誤訊息。CORS 全開放，可由任何網域呼叫。
