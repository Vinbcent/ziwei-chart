隱私權政策

紫微斗數排盤服務（ziwei-chart）　·　最後更新：2026 年 6 月

一、服務說明

本服務提供紫微斗數排盤之計算 API 與網頁工具。使用者（或代為呼叫的 AI 助理）透過 API 傳入出生資料以取得命盤結果。本服務為個人非商業性質之工具。

二、我們處理的資料

  • 排盤所需之輸入資料：出生年月日、出生時辰、性別、曆別、以及運限查詢的目標日期／時辰。
  • 這些資料僅以網址查詢參數的形式傳入，即時計算後立即回傳命盤結果，伺服器端不主動建立資料庫、不長期保存、不出售、不分享給第三方。

三、第三方與技術紀錄

  • 本服務部署於 Vercel 平台。如同一般網站，主機供應商可能基於維運與防濫用目的，於其伺服器日誌中暫時保留請求紀錄（含請求網址與來源 IP），其保存與處理依 Vercel 隱私權政策 辦理。
  • 當你透過 ChatGPT、Gemini 等 AI 平台之功能呼叫本 API 時，你與該 AI 平台之間的互動另受該平台之隱私權政策規範。

四、Cookie

本服務不使用追蹤型 Cookie。網頁工具可能於你的瀏覽器本機儲存（localStorage）暫存設定，該資料僅存在你的裝置上。

五、免責聲明

本服務之排盤與任何衍生之命理分析僅供研究與娛樂用途，不構成醫療、法律、投資或其他專業建議。

六、聯絡方式

如對本政策有疑問，可透過專案頁面 GitHub 聯繫。
